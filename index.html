How to build Web Page

Introduction

Nowadays everyone needs a web page. Wether it's for your business, school, exposition, shopping, etc. Web development can look scary at first sight, but really isn't.

In this short technical documentation i'll show you the basics that you need to start.

It's important that you understand the holy trinity of Web Development:

  • HTML: This is the vertebral column, it gives the basic structure of the page.
  • CSS: This is what gives the web it's style, like colors, sizes, bordes, decorations, etc.
  • JavaScript: This is the page soul. With this you can make the page dinamic with animations.

Basic structure

The basic structure of an HTML has a set of elements with certain purpouses.

Every HTML has to start with a DOCTYPE declaration to tell the navegator that the page is built in HTML5, wich is the newest version. The way you do it like this:

< !DOCTYPE html >

Then you have the html itself, with an opening and closing tags, like this:

< html > < / html >

Then, inside the html tags, you have to nest two more tags:

< head > < / head >

< body > < / body >

The content inside the head element is not visible for the user, but it contains important information for the navegator about the page, like it's title, the tipe of codification, links to stylesheet, etc.

The body tag contains all the visible content of the page.

Lastly, inside the body tag, we'll divide our content in three containers:

< header > < / header >

< main > < / main >

< footer > < / footer >

Inside the header tag we usually put stuff like the title and navigation bar.

Inside the main tag we usually put main content of the page, like subtitles, articles, images, videos, links, etc.

Lastly, inside the footer tag we usually put contact info, addresses, author's name, links to social media, etc.

Order

To keep an understandable order of the html structure, its important that we indent those tags that are sons of their parents tags (that are inside other) this is called NESTING.

CSS

Css is the way we style our web page. In here we tell the page how to show us the content, what sizes, what colors, what styles.
